## Code Quality Rules

### 컴포넌트 재사용
- 같은 UI/로직이 2번 이상 나오면 컴포넌트로 분리
- 기존 컴포넌트 확인 후 새로 만들기 (중복 생성 금지)

### 하드코딩 금지
- 문자열, 숫자, URL 직접 코드에 박지 않기
- 상수는 constants/ 또는 설정 파일로 분리
- 환경별 값은 환경변수 사용

### UTF-8 인코딩
- 모든 파일 UTF-8 (BOM 없음)
- 파일 읽기/쓰기 시 utf-8 명시
